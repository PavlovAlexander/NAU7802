# ESP32 NAU7802 Precision Scale Firmware

Прошивка для высокоточных весов на базе ESP32 и 24-битного АЦП NAU7802 с тензодатчиком.

## Характеристики

- **Точность**: ±2-4 мг при 30g
- **Диапазон**: 0-30g
- **Частота измерений**: ~5 Hz (200 мс на измерение)
- **Калибровка**: 4-точечная линейная регрессия (0g, 10g, 20g, 30g)
- **Интерфейс**: Serial консоль (115200 baud)

## Оптимальная конфигурация

- **Sample Rate**: 80 SPS
- **Gain**: 128x
- **I2C Clock**: 100 kHz (стабильность)
- **Chopper**: Отключён (CLK_CHP = 0b11)
- **Усреднение**: 16 отсчётов

## Быстрый старт

### Требования

- PlatformIO
- ESP32 DevKit v1
- NAU7802 breakout board (SparkFun или аналог)
- Тензодатчик (load cell)
- Эталонные гирьки (0g, 10g, 20g, 30g)

### Подключение

```
ESP32          NAU7802
GPIO 21 (SDA) → SDA
GPIO 22 (SCL) → SCL
3.3V          → VCC
GND           → GND
```

### Сборка и загрузка

```bash
pio run -t upload
pio device monitor
```

### Калибровка

1. При первом запуске нажмите ENTER для калибровки
2. Последовательно размещайте гирьки 0g, 10g, 20g, 30g
3. Нажимайте ENTER после каждой гирьки
4. Калибровка сохраняется в NVS автоматически

### Команды консоли

- `C` — Запустить калибровку
- `S` — Показать статус калибровки
- `R` — Перезагрузка системы
- `L` — Загрузить сохранённую калибровку (при старте)
- `N` — Новая калибровка (при старте)

## Планируемые улучшения

Спецификация программных улучшений точности находится в `.kiro/specs/scale-accuracy-improvements/`:

### 1. Нелинейная калибровка
- Кусочно-линейная интерполяция между 4 точками
- Полиномиальная регрессия 2-3 степени
- Улучшение точности во всём диапазоне

### 2. Температурная компенсация
- Использование встроенного температурного сенсора NAU7802
- Коррекция коэффициентов k и b при изменении температуры
- Двухточечная калибровка для вычисления температурных коэффициентов

### 3. Фильтрация выбросов
- Медианный фильтр (окно 3/5/7)
- Статистический 3σ фильтр
- Отбрасывание аномальных значений от I2C ошибок

### 4. Адаптивное усреднение
- Больше отсчётов при стабильном весе (точность)
- Меньше отсчётов при изменении веса (скорость отклика)
- Автоматический баланс на основе σ

**Статус**: Требования и проектирование завершены, готов план реализации (tasks.md)

### Test Wizard — Интерактивное тестирование

Для удобного тестирования всех улучшений реализован **Test Wizard** — интерактивный мастер, который:
- Проводит пользователя через 6 автоматизированных тестов
- Автоматически собирает данные и вычисляет статистику
- Генерирует подробный отчёт с PASS/FAIL индикаторами
- Экспортирует результаты в CSV для анализа

**Команды**:
```
test wizard start    - Запустить мастер тестирования
test wizard report   - Показать отчёт
test wizard export   - Экспорт в CSV
```

Подробности в [TEST_WIZARD_GUIDE.md](.kiro/specs/scale-accuracy-improvements/TEST_WIZARD_GUIDE.md)

## Известные проблемы

1. **I2C ошибки (Error 263)**: Возникают при приближении руки к устройству (электромагнитные помехи). Решение: экранирование проводов.
2. **Выбросы при изменении веса**: Механические вибрации. Будет решено фильтром выбросов.
3. **Температурный дрейф**: ~0.5-1 мг при ΔT=10°C. Будет решено температурной компенсацией.
