---

## 2. NAU7802 — используемые режимы чипа

| Параметр | Значение |
|----------|----------|
| Скорость дискретизации | **`CRS_80`** (номинально **~80 выборок/с**) |
| PGA | **`PGA_128`** |
| LDO (настройка чипа для моста) | В `begin()` библиотеки: **`LDO_3_3V`** (`setConfig`) |
| `PU_CTRL[7]` AVDDS | **Включено** после `begin()`: внутренний AVDD (`enableInternalAvdd()`, `PU_CTRL \| 0x80`, пауза **300 ms**) |
| `PWR_CTRL` (0x1C) | Задаётся библиотекой в `begin()`: **0x80** (PGA cap по комментарию библиотеки) |
| `ADC_CTRL3` (0x1B) | Задаётся библиотекой в `begin()`: **0x40** |
| Непрерывные преобразования | Библиотека: `startReading()` (бит CS), опрос готовности через `available()` / `readRaw()` |

---

## 3. Чоппер (ADC_CTRL1, 0x15)

| Параметр | Значение |
|----------|----------|
| REG_CHPS [5:4] | **0b11** (биты **0x30**), библиотека выставляет в `begin()`, прошивка дополнительно нормализует в `initChopperControl()` |
| REG_CHP [1:0] | Режим задержки **0…3**, по умолчанию после init — **0**; рантайм: клавиша **`J`** (цикл), **`U`** — дамп регистров |

---

## 4. Усреднение и таймауты чтения (библиотека 7Semi)

Библиотека в `readAverage(numSamples, maxTimeout)` **делит** `maxTimeout / numSamples` на каждый внутренний `readRaw` и ограничивает общее время цикла.

| Режим | Усреднение | maxTimeout | Примечание |
|--------|------------|--------------|------------|
| Взвешивание | **16** | **4000 ms** | до **4** повторов с `delay(5)`; при **3** подряд полных неудачах — **`initHardware()`** |
| Калибровка (точка) | **50** | **2000 ms** | после Enter |
| Flush при init | 6× `readRaw` | **250 ms** на вызов | после инициализации |

---

## 5. Калибровка и NVS

| Параметр | Значение |
|----------|----------|
| Модель | **PIECEWISE_LINEAR** |
| Точки | **0, 10, 20, 30 g** |
| Защита от насыщения | \|raw\| ≥ **8388000** → ошибка |
| Мин. span raw (0 g vs 30 g) | ≥ **10000** LSB; иначе калибровка не сохраняется / не загружается из NVS |
| Хранение | `Preferences`, namespace **`scale_cal`** |

---

## 6. Сглаживание и отображение (программно, граммы)

Реализация: `scale_weighing_7semi.cpp` (после `piecewiseWeightFromRaw`).

| Параметр | Значение |
|----------|----------|
| Быстрое сглаживание α | **0.70**, если \|Δ\| > **30 mg** (`WEIGH_BIG_DELTA_G`) |
| Медленное сглаживание α | **0.15** |
| Deadband на выводе | **0.002 g** (2 mg) |

---

## 7. Тарирование (zero)

| Параметр | Значение |
|----------|----------|
| Способ | **Программный** offset в граммах (`g_tareOffset`), клавиша **`0`** в режиме взвешивания |
| Периодическое авто-zero | **Не реализовано** (только вручную **`0`**) |
| Библиотечный `tare()` по raw | **Не используется** в текущем FSM |

---

## 8. Питание (железо)

Код не задаёт напряжение на разъёме платы: фактически **USB → регулятор 3.3 V ESP32** и питание модуля NAU7802 по вашей разводке. В регистрах NAU7802 включены **внутренний AVDD (AVDDS)** и **LDO 3.3 V** для типовой схемы с мостом.

## 11. Клавиши консоли (кратко)

| Клавиша | Действие |
|---------|----------|
| `H` | Справка |
| `L` / `N` | Загрузка калибровки / новая калибровка |
| `Q` | Выход из взвешивания |
| `0` | Таре (zero) |
| `J` | Цикл REG_CHP (чоппер delay) |
| `U` | Дамп регистров NAU7802 |
| `S` | Статус калибровки |
| `R` | Перезапуск ESP32 |
