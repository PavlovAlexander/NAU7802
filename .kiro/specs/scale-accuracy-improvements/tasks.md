# План реализации: Улучшение точности весов ESP32 + NAU7802

## Обзор

Данный план описывает пошаговую реализацию четырёх независимых компонентов для повышения точности весов с текущих ±2-4 мг до субмиллиграммового уровня:

1. **Нелинейная калибровка** — кусочно-линейная интерполяция и полиномиальная регрессия
2. **Температурная компенсация** — коррекция температурного дрейфа через встроенный сенсор NAU7802
3. **Фильтрация выбросов** — медианный и статистический 3σ фильтры
4. **Адаптивное усреднение** — автоматический баланс точность/скорость

Каждый компонент может быть включён/отключён независимо. Реализация включает 14 property-based тестов для математических алгоритмов.

## Примечания

- Задачи, помеченные `*`, являются опциональными и могут быть пропущены для более быстрого MVP
- Каждая задача ссылается на конкретные требования для отслеживаемости
- Checkpoints обеспечивают инкрементальную валидацию
- Property-based тесты валидируют универсальные свойства корректности (минимум 100 итераций каждый)
- Unit-тесты валидируют конкретные примеры и граничные случаи
- Интеграционные тесты требуют физического оборудования (ESP32 + NAU7802)

## Порядок выполнения

Рекомендуется выполнять задачи последовательно, так как каждая задача строится на предыдущих. Компоненты независимы, но интеграция требует наличия всех базовых модулей.

Для начала выполнения откройте этот файл и нажмите "Start task" рядом с задачей.
